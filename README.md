# ng-blog

Mini app for personal blogging in Angular.

## Features

- User registration and login.
- Public posts and posts available by request.
- Follow users and build a feed from subscriptions.
- Create, edit, and delete personal posts.
- Sort and filter posts by tags.
- Comment on posts.

## Run

Install dependencies:

```bash
npm install
```

Start JSON Server:

```bash
npm run api
```

Start Angular:

```bash
npm start
```

Open `http://localhost:4200/`.
